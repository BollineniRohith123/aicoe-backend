## How to Use for Testing

1. **Upload any of these files** to your AICOE platform as a meeting transcript
2. **Run the full pipeline** to generate:
   - Structured notes extraction
   - Use case identification
   - PRD generation
   - Mockup creation (with AICOE branding)
   - Synthetic data generation
   - Knowledge base enrichment

3. **Verify outputs** match:
   - AICOE color scheme (deep navy backgrounds, white text, bright blue accents)
   - Apple-style design principles
   - Lucid icon usage
   - Responsive design
   - Industry-specific requirements

## Testing Scenarios

### Comprehensive Pipeline Test
- **Input:** Any meeting transcript
- **Transcript Agent:** Extracts structured notes, attendees, objectives, action items
- **Requirements Agent:** Identifies use cases, business requirements, technical constraints
- **PRD Agent:** Generates complete product requirements document
- **Mockup Agent:** Creates AICOE-branded HTML mockups with proper styling
- **Synthetic Data Agent:** Produces sample data matching use case requirements
- **Knowledge Base Agent:** Enriches content with industry patterns and best practices
- **Reviewer Agent:** Provides feedback interface with AICOE branding
- **Storage Agent:** Organizes outputs in proper project folder structure

- **Colors:** Verify deep navy backgrounds, white text, bright blue/cyan accents

- **Consistency:** Validate branding across all generated artifacts
