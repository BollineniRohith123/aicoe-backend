Use Cases

UC-001 High Priority

Perform Basic Addition

User adds two or more numbers using the calculator's addition functionality

Actors

  • End User

Preconditions

  • Calculator application is open and displayed
  • All number and operation buttons are functional

Main Flow

  • User enters the first number using numeric keypad
  • User presses the addition (+) operator button
  • User enters the second number using numeric keypad
  • User can optionally enter more numbers separated by addition operators
  • User presses the equals (=) button
  • Calculator displays the sum of all entered numbers

Alternate Flows

Clear Entry
  • User presses clear (C) button at any point
  • Calculator resets to zero
  • User can start new calculation
Decimal Numbers
  • User can enter decimal points in numbers
  • Calculator handles floating-point arithmetic correctly

Postconditions

  • Result is displayed on screen
  • Calculator is ready for new calculation or can use result for further operations

Business Value

Core functionality that enables basic arithmetic calculations, meeting the primary objective of the calculator application

UC-002 High Priority

Perform Basic Subtraction

User subtracts numbers using the calculator's subtraction functionality

Actors

  • End User

Preconditions

  • Calculator application is open and displayed
  • All number and operation buttons are functional

Main Flow

  • User enters the first number (minuend) using numeric keypad
  • User presses the subtraction (-) operator button
  • User enters the second number (subtrahend) using numeric keypad

  • Calculator displays the difference between the numbers

Alternate Flows

Multiple Subtractions
  • User can chain multiple subtractions
  • Each subtraction is performed in sequence
  • Final result shows cumulative difference

Postconditions

  • Result is displayed on screen
  • Calculator maintains state for potential further calculations

Business Value

Essential arithmetic operation required for basic calculator functionality

UC-003 High Priority

Perform Basic Multiplication

User multiplies two or more numbers using the calculator's multiplication functionality

Actors

  • End User

Preconditions

  • Calculator application is running
  • Multiplication button is available and functional

Main Flow

  • User enters the first number using numeric keypad

  • User enters the second number using numeric keypad
  • User can optionally enter more numbers separated by multiplication operators
  • User presses the equals (=) button

UC-005 Medium Priority

Clear Calculator Display

User clears the calculator display and resets calculations

Actors

  • End User

Preconditions

  • Calculator is displaying a number or result
  • Clear button is available

Main Flow

  • User presses the Clear (C) button
  • Calculator display resets to zero
  • All pending operations are cancelled
  • Calculator is ready for new input

Alternate Flows

Clear Entry vs Clear All
  • Clear Entry (CE) removes last entry only
  • Clear (C) resets entire calculator

Postconditions

  • Calculator display shows zero
  • No pending operations remain
  • Ready for new calculation

Business Value

Provides user control and error correction capability, improving user experience
