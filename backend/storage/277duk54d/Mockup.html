Use Cases & Workflows

UC-001 High Priority

Log a Workout

The user records the details of a completed workout session, including exercises, sets, reps, weight, and duration, to maintain a historical record of their physical activity.

Preconditions
  • The user is logged into the application
  • The user has completed a workout they wish to log
Main Flow
  • Navigate to 'Workout' section
  • Select 'Log New Workout'
  • Enter workout details (date, exercises, sets, reps, weight)
  • Review workout summary
  • Save workout to history
Alternate Flows
  • Use Workout Template for quick logging
  • Cancel logging without saving
UC-002 High Priority

View Fitness Progress

The user visualizes their performance over time through charts and graphs to understand trends, stay motivated, and make informed adjustments to their fitness plan.

Preconditions
  • The user is logged into the application
  • The user has logged at least one workout
Main Flow
  • Navigate to 'Progress' or 'Dashboard' section
  • View default overview (last 30 days)
  • Select specific metrics to visualize
  • Adjust time frame as needed
  • View dynamic charts and graphs
Alternate Flows
  • No data available - prompt to log first workout
  • Filter progress by specific exercise
UC-003 High Priority

Set a Fitness Goal

The user defines specific, measurable fitness objectives to provide direction and purpose to their training, with the app helping to track completion.

Preconditions
  • The user is logged into the application
Main Flow
  • Navigate to 'Goals' section
  • Select 'Create New Goal'
  • Choose goal template or create custom
  • Enter target parameters
  • Save goal to active goals list
Alternate Flows
  • Edit existing goal parameters
  • Delete goal with confirmation
UC-004 Medium Priority

Track Daily Nutrition

The user records their food and beverage intake to monitor caloric consumption and macronutrient balance, complementing their fitness efforts.

Preconditions
  • The user is logged into the application
Main Flow
  • Navigate to 'Nutrition' section
  • Select meal type (Breakfast, Lunch, Dinner, Snacks)
  • Search or scan food items
  • Specify serving size
  • Add to meal and update daily totals
Alternate Flows
  • Scan barcode for automatic lookup
  • Create custom food entry
UC-005 Low Priority

Share an Achievement

The user shares a fitness milestone, such as a new personal record or a completed goal, with their social network or within the app community to celebrate success and foster motivation.

Preconditions
  • The user is logged into the application
  • The user has recently achieved a notable milestone
Main Flow
  • System presents 'Congratulations' screen
  • Click 'Share' button
  • Select sharing destination
  • Review auto-generated achievement graphic
  • Add personal comment and confirm share
Alternate Flows
  • Decline to share (achievement remains private)
  • Share from history at any time
