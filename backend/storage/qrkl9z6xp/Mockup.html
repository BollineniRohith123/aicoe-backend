Use Cases

UC-001

View Real-Time Dashboard

High Priority

The user logs into the system and views their primary or a selected analytics dashboard. The dashboard displays a collection of widgets showing live metrics, KPIs, and data visualizations that update in real-time as new data becomes available.

Actors
User, Business Analyst, Manager
Preconditions
Authenticated, Dashboard configured, Data sources operational
Key Flows
Navigate, Authenticate, Load widgets, WebSocket connection
Business Value

Provides immediate visibility into business performance and operational health, enabling faster, data-driven decision-making and proactive issue detection.

UC-002

Customize Dashboard Layout

High Priority

The user modifies the layout and content of their dashboard to suit their specific needs. This includes adding new widgets, removing existing ones, resizing them, and rearranging their position on the grid.

Actors
User
Preconditions
Logged in, Edit permissions
Key Flows
Edit mode, Drag widgets, Resize, Save changes
Business Value

Empowers users to create personalized views that are most relevant to their roles and responsibilities, increasing user adoption, engagement, and the overall utility of the dashboard.

UC-003

Create and Manage User Dashboards

Medium Priority

The user creates new, blank dashboards for specific purposes, and can manage their existing dashboards by renaming them, deleting them, or setting one as the default.

Actors
User
Preconditions
Logged in
Key Flows
Access management, Create dashboard, Rename, Delete, Set default
Business Value

Allows for better organization of analytics by enabling users to build dedicated dashboards for different projects, departments, or business questions, improving clarity and focus.

UC-004

Export Dashboard Data

Medium Priority

The user exports the data or visualizations from a dashboard for use in external reports, presentations, or for further offline analysis.

Actors
User
Preconditions
Logged in, Export permissions
Key Flows
Export button, Format selection, Download file
Business Value

Facilitates the sharing of insights with stakeholders who may not have direct access to the dashboard and enables integration of real-time data into static reporting and archival processes.
