UC-004

Select and Customize Workout Plan

High Priority

A beginner user browses and selects a pre-made workout plan tailored to their fitness level and goals, with options for customization.

Actors

Beginner User

Preconditions

User has completed onboarding and fitness level assessment

Postconditions

User is enrolled in a structured workout program with clear path and scheduled sessions

Key Features

Duration
30 Days
Difficulty
Beginner
Equipment
Minimal
Target Users
New to Fitness

Main Flow

  • User navigates to 'Workout Plans' section
  • App displays curated plans for beginners (e.g., '30-Day Challenge', 'Home Workout Basics')
  • Each plan shows duration, difficulty, equipment needed, and preview of exercises
  • User taps on a plan to view detailed schedule and progression
  • App shows user testimonials and success stories for social proof
  • User selects 'Start Plan' and can customize start date
  • App integrates the plan into the user's calendar and sets up reminders
  • User receives confirmation and first day's workout is scheduled

Alternate Flows

User can modify the plan by swapping exercises. This premium feature allows customization based on preferences or equipment availability.

User can pause or restart the plan at any time. Progress is saved and can be resumed from where the user left off.

If user has equipment limitations, app suggests alternative plans that match available equipment and fitness goals.
