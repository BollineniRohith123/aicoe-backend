UC-001

Log a Workout Session

The user manually logs the exercises, sets, and reps completed during a workout session, either as part of a pre-made plan or a custom workout. This use case is designed to be simple and quick to accommodate beginners.

Actors

User (Fitness Beginner)

Priority

High Priority
User is logged into the application.
User has either selected a pre-made workout plan or chosen to log a custom workout.
1
User navigates to the 'Log Workout' screen.
2
System displays the current day's planned exercises if a plan is active, or presents an option to 'Add Exercise'.
3
User selects an exercise from the list (e.g., 'Squat').
4
System displays the exercise demonstration (image/GIF/link) and input fields for sets, reps, and weight.
5
User enters the number of sets, reps, and weight lifted for the first set.
6
User taps 'Log Set'.
7
System saves the set and presents input fields for the next set.
8
User repeats steps 5-7 for all sets of the exercise.
9
User taps 'Complete Exercise' to finish the current exercise.
10
System marks the exercise as complete and returns the user to the workout overview.
11
User repeats steps 3-10 for all exercises in the workout.
12
User taps 'Finish Workout'.
13
System displays a summary screen with total volume, estimated calories, and a congratulatory message.
14
System awards a badge or streak milestone if applicable (gamification).
15
User taps 'Save Workout'.
16
System saves the complete workout session to the user's history and updates progress visualizations.

Add Custom Exercise

If the desired exercise is not in the list, the user can search for it or add a new custom exercise with a name and muscle group.

Use Previous Stats

When logging an exercise done before, the system pre-fills the weight and reps from the last session to save the user time.

The workout session is saved and accurately reflected in the user's progress history.
Progress charts and achievements are updated.
Data is ready for synchronization with Apple Health/Google Fit.

Business Value Analysis

This is a core feature that provides the fundamental value of the app. A simple, frictionless logging experience is critical for user retention and differentiates the app from more complex competitors, directly addressing the needs of the target beginner audience.

Key Benefits:

Increases user engagement and retention
Caters specifically to fitness beginners
Provides competitive differentiation
Reduces friction in workout logging

Technical Considerations

Local storage for offline logging capabilities
Background sync with cloud storage
Responsive design for mobile-first experience
Secure data handling for personal fitness information
