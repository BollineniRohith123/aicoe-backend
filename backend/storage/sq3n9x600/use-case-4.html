UC-004

Select and Start a Pre-made Workout Plan

A new user browses a library of beginner-friendly workout plans, selects one that fits their goals (e.g., 'Get Started', 'Build a Habit'), and begins it. This reduces the cognitive load for beginners who don't know what to do.

Actors

User (Fitness Beginner)

Priority

High

Preconditions

User is logged into the application.

Postconditions

A workout plan is activated for the user.
The user's home/log workout screen is updated to reflect the scheduled workouts from the new plan.

Main Success Scenario

  • User

    navigates to the 'Plans' tab.

  • System

    displays a curated list of workout plans for beginners, each with a title, duration (e.g., '4 weeks'), and goal (e.g., 'Full Body Strength').

  • User

    scrolls through the plans. Some plans are marked 'Premium'.

  • User

    taps on a free plan, for example, '2-Week Intro to Fitness'.

  • System

    displays a detailed overview of the plan: description, weekly schedule, list of exercises, and required equipment.

  • User

    taps the 'Start Plan' button.

  • System

    presents a confirmation modal: 'Starting this plan will set your workout schedule. Are you ready?'

  • User

    taps 'Confirm'.

  • System

    activates the plan. The 'Log Workout' screen now shows today's scheduled workout from the plan.

  • System

    navigates the user to the newly activated plan's main screen, showing Day 1 of Week 1.

Alternate Flows

Select a Premium Plan

If the user taps on a 'Premium' plan and then 'Start Plan', the system displays the subscription upgrade screen, highlighting the benefits of premium, including personalized plans.

Pause or Switch Plan

From the active plan screen, the user can choose to 'Pause Plan' or 'Switch Plan', allowing for flexibility in their fitness journey.

Business Value

This feature is essential for the target audience of beginners who feel intimidated. It provides immediate structure and guidance, lowering the barrier to entry and preventing user churn. It also serves as a key hook for the freemium model by showcasing more advanced, personalized plans behind the paywall.
