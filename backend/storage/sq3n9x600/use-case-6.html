Manage Notification Preferences UC-006

Fitness Tracking App - Chrome MCP Test

Description

The user navigates to settings to customize the types and frequency of push notifications they receive, ensuring they feel motivated and not spammed, which aligns with user expectations for control.

Business Value

Key Benefits

Giving users granular control over notifications is a best practice for user retention and respecting user privacy. It directly mitigates the risk of annoying users with push notifications, a key pain point identified, and builds long-term trust.

Notification Categories

Workout Reminders
Daily exercise prompts
Streak Celebrations
Achievement notifications
Progress Updates
Weekly/monthly reports
Premium Offers
Marketing notifications

Actors

User (Fitness Beginner)

A new user to the fitness tracking application who is just starting their fitness journey.

Preconditions

  • User is logged into the application.

Postconditions

  • The user's notification preferences are updated.
  • The system will only send push notifications that align with the user's chosen settings.

Main Flow

  • User navigates to the 'Settings' screen.
  • User taps on 'Notifications'.
  • System displays a list of notification categories with toggles.
  • Categories include: 'Workout Reminders', 'Streak Celebrations', 'Progress Updates', 'Premium Offers'.
  • User taps on 'Workout Reminders'.
  • System allows the user to toggle the feature on/off and select the days of the week and time of day for the reminder.
  • User sets their reminder for weekdays at 6:00 PM.
  • User navigates back to the main Notifications screen.
  • User toggles off 'Premium Offers' because they find them annoying.
  • System saves the user's preferences immediately.

Alternate Flows

OS-level Notification Disabled

If the user has disabled notifications for the app at the OS level, the system displays a message with instructions on how to re-enable them in the device's main settings.
