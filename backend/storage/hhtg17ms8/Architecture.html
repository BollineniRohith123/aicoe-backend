System Components

Detailed breakdown of all system components and their technologies

ComponentTypeTechnologyDescription
Mobile Application React Native App React Native, Redux, SQLite Cross-platform mobile app for iOS and Android with offline-first capabilities
API Gateway Gateway Service AWS API Gateway / Kong Central entry point for all API requests with authentication and rate limiting
Workout Service Microservice Node.js / Express.js Handles workout logging, exercise database, and workout plan management
Nutrition Service Microservice Python / FastAPI Manages nutrition logging with AI-powered food recognition
ML Service AI/ML Service TensorFlow, SageMaker, Python Machine learning service for personalization and computer vision
User Service Microservice Spring Boot / Java Manages user profiles, authentication, and preferences
Achievement Service Microservice Node.js Gamification engine for badges, streaks, and achievements
Plan Service Microservice Node.js Manages workout plans and scheduling

External Integrations

Third-party services and APIs integrated into the system

  • Apple HealthKit (Native SDK Integration): Sync health and workout data from Apple ecosystem
  • Google Fit (REST API Integration): Aggregate health data from Android devices and Google services
  • Wearable Devices (Bluetooth/Web API): Connect with Fitbit, Garmin, and other fitness trackers
  • Spotify API (OAuth 2.0 API): Integrate workout playlists and music control
  • Calendar APIs (REST API): Schedule workouts and send reminders
  • Google Vision API (Cloud AI Service): Food recognition for nutrition logging
  • Payment Gateway (Payment API): Handle premium subscriptions and in-app purchases

Hosting and infrastructure details

Hosting: AWS (EKS, Lambda, EC2)
