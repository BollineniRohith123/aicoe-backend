UC-002

Start a Pre-made Workout Plan

A user browses, selects, and begins a structured, pre-made workout plan designed for beginners (e.g., '30-Day Full Body Challenge'). This provides guidance and structure, reducing the intimidation factor for new users.

Actors
  • Registered User
Preconditions
  • User is logged into the app.
Postconditions
  • A workout plan is active for the user.
  • The user's dashboard is updated to reflect the current plan and progress.
  • Scheduled workout reminders are set.
Priority
High Priority Critical for user onboarding and retention
Main Flow
  1. User navigates to the 'Plans' section of the app.
  2. The app displays a curated list of workout plans, filtered for beginners. Plans are clearly marked as 'Free' or 'Premium'.
  3. User taps on a plan to view details, including duration, frequency, target muscle groups, required equipment, and a preview of the first week's workouts.
  4. User taps the 'Start Plan' button.
  5. If the plan is free, the plan is immediately activated and added to the user's profile.
  6. The user is navigated to the 'Today's Workout' screen, which shows the first scheduled workout.
  7. The app sends a confirmation notification and sets up reminder notifications for the plan's schedule.
Alternate Flow: Selecting a Premium Plan

If the user taps 'Start Plan' on a premium plan without an active subscription, they are redirected to the subscription purchase screen. Upon successful payment, the plan is activated.

This flow requires integration with payment processing and subscription management.
Alternate Flow: Switching Plans

If a user already has an active plan and selects a new one, the app prompts them: 'Starting a new plan will pause your current one. Continue?'. Upon confirmation, the new plan is activated.

Important Considerations:
  • • Preserve progress data from paused plan
  • • Allow easy resume of paused plans
  • • Clear communication about plan status

Full Description

A user browses, selects, and begins a structured, pre-made workout plan designed for beginners (e.g., '30-Day Full Body Challenge'). This provides guidance and structure, reducing the intimidation factor for new users. The feature serves as a critical entry point for users who may be overwhelmed by creating custom workouts from scratch, offering them expert-designed programs that ensure proper progression and variety.

Business Value

Provides immediate value and structure for the target beginner audience. This is a key driver for the freemium model, as high-quality plans are a primary reason for users to upgrade to a premium subscription, a practice validated by market leaders like Peloton and Nike Training Club.
