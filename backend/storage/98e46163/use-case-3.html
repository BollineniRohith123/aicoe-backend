UC-003

User Tracks a Meal Using AI Photo Recognition

The user wants to log their food intake easily. They use the device's camera to take a photo of their meal, and the app uses a third-party AI API to identify the food items and estimate nutritional information (calories, macros). The user can then review and edit the results.

Priority: High
Page 3 of 6
Created: 2024-01-15

Actors

Primary User (Fitness Beginner)

Preconditions

  • User has completed onboarding.
  • User has granted the app permission to access the device's camera.

Main Flow

  1. User navigates to the 'Nutrition' tab.
  2. User taps the 'Track Meal' button.
  3. User selects the 'Photo' option.
  4. App opens the device camera interface.
  5. User takes a photo of their meal.
  6. App sends the photo to a third-party AI nutrition API for analysis.
  7. App displays a loading indicator while processing.
  8. The API returns a list of identified food items with estimated portion sizes and nutritional information.
  9. App presents the results to the user for confirmation.
  10. User reviews the identified items. They can tap an item to view details, adjust the portion size using a slider, or delete an incorrect item.
  11. User can add missing items manually using a search bar.
  12. User taps 'Confirm & Log'.
  13. The final nutritional data (calories, protein, carbs, fat) for the meal is saved to the user's daily log.

Alternate Flows

Condition:

AI fails to recognize the food or returns highly inaccurate results.

Flow:

The app displays a message 'Couldn't identify food. Please log manually.' and presents the manual food search interface. This mitigates the risk of AI inaccuracy.

Condition:

User prefers manual entry.

Flow:

From the 'Track Meal' screen, the user can select the 'Search' option to find and add food items from a database directly, bypassing the photo feature.

Postconditions

  • A meal and its associated nutritional data are successfully logged in the user's daily nutrition summary.
  • The user's daily calorie and macro progress bars are updated.

Business Value

Key Differentiator

This is a key differentiator aligned with the industry trend of AI-powered personalization. It directly addresses the competitor insight that apps like MyFitnessPal are complex, by offering a much simpler alternative. It fulfills a core user expectation for easy and accurate nutrition logging.
