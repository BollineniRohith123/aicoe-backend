UC-004

User Views Progress and Achievements

The user accesses their personal dashboard to visualize their fitness and nutrition progress over time. This includes charts for weight, body measurements, and workout performance, as well as a section for unlocked badges and achievements to provide motivation.

Priority: Medium
Page 4 of 6

Actors

Primary User (Fitness Beginner)

Summary

This use case enables users to track and visualize their fitness journey through comprehensive progress charts and achievement badges. By providing visual feedback on weight trends, workout performance, and nutritional data, users can stay motivated and informed about their progress. The gamification element through achievements adds an extra layer of engagement, encouraging consistent app usage and goal achievement.

Main Flow

  1. User navigates to the 'Progress' tab from the main dashboard.
  2. App displays a summary view with key metrics: current weight, workouts completed this month, and a daily calorie/macro overview.
  3. User taps on the 'Weight' chart.
  4. App displays a line graph showing the user's weight entries over time.
  5. User taps on the 'Workout Performance' section.
  6. App displays visualizations of key strength indicators, such as the total volume lifted for major exercises (e.g., squat, deadlift) over time.
  7. User navigates to the 'Achievements' section within the Progress tab.
  8. App displays a grid of locked and unlocked badges (e.g., 'First Workout', '7-Day Streak', '5K Runner').
  9. User taps on a locked badge to see the requirements for unlocking it.
  10. User taps on an unlocked badge to see the date it was earned.

Alternate Flows

Condition: User has no data to display

The app displays an encouraging message: 'Start logging workouts and meals to see your progress here!' with quick links to the 'Workout' and 'Nutrition' tabs.

Condition: A new achievement is unlocked

The next time the user views the Progress tab or opens the app, a celebratory modal appears, congratulating them on unlocking the new badge and explaining its significance.

Preconditions

  • User has logged at least one workout or meal.
  • User has been using the app for a period of time (e.g., one week).

Postconditions

  • User has a clear understanding of their progress and trends.
  • User feels motivated by their visualized achievements and is aware of future goals.

Business Value

Progress visualization is a critical feature for user motivation and long-term retention, a key user expectation. Incorporating gamification through achievements addresses the industry trend of boosting engagement and provides a clear value proposition for the premium tier in the freemium model.
