Chat Pattern Guidelines

  • Message Alignment: AI messages left-aligned, user messages right-aligned
  • Avatars: Bot icon for AI, user icon for humans (32px circular)
  • Timestamps: Use caption text size, secondary color
  • Actions: Provide quick action buttons for common tasks
  • Typing Indicator: Use pulse animation to show AI is thinking
  • Input: Always accessible at bottom with clear send button
  • Max Width: Limit message bubbles to 70% of container width for readability
  • Scrolling: Auto-scroll to latest message, maintain scroll position when new messages arrive

Tab Pattern Guidelines

  • Active State: Clearly indicate which tab is currently selected with border/background
  • Icons: Use Lucide icons for visual identification (optional but recommended)
  • Layout: Horizontal tabs for main navigation, vertical for sidebars
  • Content: Only one tab content visible at a time
  • Accessibility: Use proper ARIA labels and keyboard navigation
  • Responsive: Consider dropdown menu for mobile viewports

Visualization Pattern Guidelines

  • Color Usage: Use accent colors to highlight important data points
  • Labels: Always include axis labels and legends
  • Tooltips: Add hover tooltips for detailed information
  • Accessibility: Provide text alternatives and keyboard navigation
  • Responsive: Ensure charts scale properly on different screen sizes
  • Libraries: Chart.js (simple), D3.js (advanced), Three.js (3D)
  • Loading States: Show skeleton or spinner while data loads
  • Empty States: Display meaningful message when no data available

Navigation Pattern Guidelines

Breadcrumbs
  • Show hierarchical path
  • Use chevron separators
  • Last item not clickable
  • Limit to 5-6 levels max
Sidebar Navigation
  • Active state clearly visible
  • Use icons for clarity
  • Collapsible on mobile
  • Group related items
Pagination
  • Show current page clearly
  • Include prev/next buttons
  • Show total count
  • Use ellipsis for long lists
Step Progress
  • Show completed steps
  • Highlight current step
  • Indicate remaining steps
  • Allow backward navigation
