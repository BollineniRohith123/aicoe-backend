Usage Guidelines

Do:
  • Use on light or dark backgrounds
  • Maintain minimum clear space around logo
  • Use provided size classes (.aicoe-logo-sm, .aicoe-logo-lg, etc.)
  • Include alt text for accessibility
  • Maintain aspect ratio (always square)
  • Use the PNG file provided
Don't:
  • Modify the PNG file or colors
  • Stretch or distort the logo
  • Use on busy or low-contrast backgrounds
  • Add effects, shadows, or filters
  • Recreate the logo from scratch
  • Use sizes smaller than 32px
